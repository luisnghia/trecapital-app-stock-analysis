# V23.20 - UI Tổng quan doanh nghiệp, STT và note loại doanh nghiệp

- Tổng quan doanh nghiệp dùng HTML inline để làm nổi bật mã cổ phiếu/tên doanh nghiệp, tránh CSS Streamlit bị cache.
- Thay st.metric ở vùng tổng quan bằng thẻ chỉ tiêu nhỏ gọn hơn khoảng 40%.
- Cột STT trong các bảng Công thức & audit định dạng số nguyên, không có thập phân.
- Note khi nhấp vào dòng “Doanh nghiệp thuộc loại nào?” trong tab Định giá chuyên sâu bổ sung diễn giải theo loại doanh nghiệp: Quality Compounder, Cyclical, Asset Play, Turnaround, Bank/Insurance, Normal Business.
