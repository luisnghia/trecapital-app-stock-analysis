# V23.44 – Oaktree-inspired UI refresh

## Nội dung chỉnh sửa
- Thêm `ui_oaktree_theme.py` để inject lớp CSS giao diện lấy cảm hứng từ trang Oaktree Capital.
- Chuyển giao diện sang phong cách tổ chức đầu tư: nền trắng/kem, xanh rêu đậm, điểm nhấn vàng đồng, đường kẻ mảnh, header dạng mission block.
- Chỉnh sidebar điều hướng, tab, button, card, metric, bảng dữ liệu và chart container theo phong cách tối giản.
- Chỉ thay đổi lớp hiển thị; không thay đổi logic lấy dữ liệu, định giá, Porter Moat, MOS, peer comparison hoặc xuất báo cáo.

## Lưu ý
Đây là giao diện "Oaktree-inspired" để phù hợp định hướng app đầu tư chuyên nghiệp; không sử dụng logo hoặc tài sản thương hiệu của Oaktree.
